# Contributing

Thank you for your interest in improving FocusBoard.

## Development Workflow

1. Fork the repository.
2. Create a feature branch.
3. Make a focused change.
4. Run the project locally.
5. Open a pull request with a clear description.

## Code Style

- Use clear names for classes, methods, and variables.
- Keep UI logic, model logic, and persistence logic separated.
- Prefer small, readable methods over large blocks of code.
- Keep user-facing text in English.

## Local Build

```bash
mvn clean package
```
